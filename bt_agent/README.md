# bt_agent

A general purpose [Behavior Tree](https://en.wikipedia.org/wiki/Behavior_tree_(artificial_intelligence,_robotics_and_control)) based agent.

This agent provides a [ROS2 Action](https://docs.ros.org/en/foxy/Tutorials/Understanding-ROS2-Actions.html).
It implements this action by executing a Behavior Tree.
colcon build --symlink-install --packages-select bt_agent
